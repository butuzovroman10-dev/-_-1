Point2D Статическая библиотека
Описание
Кроссплатформенная статическая библиотека на C++, реализующая класс Point2D для работы с двумерными точками. Библиотека предоставляет функционал для выполнения геометрических операций с точками в 2D пространстве.

Требования
CMake версии 3.10 или выше

Компилятор C++ с поддержкой стандарта C++11 или выше:

GCC (Linux/macOS)

MinGW (Windows)

MSVC (Windows)

Clang

Функциональность класса Point2D
1. Приватные поля
x - координата по оси X

y - координата по оси Y

2. Конструкторы
Point2D() - конструктор по умолчанию (создает точку в начале координат)

Point2D(double x, double y) - конструктор с параметрами

3. Get и Set методы
getX(), getY() - получение координат

setX(double), setY(double) - установка координат

4. Public методы
distanceTo(const Point2D& other) - расстояние до другой точки

move(double dx, double dy) - перемещение точки

rotate(double angle) - поворот точки на заданный угол

isOrigin() - проверка, находится ли точка в начале координат

distanceFromOrigin() - расстояние от начала координат

reflectOverX() - отражение относительно оси X

5. Перегруженные операторы
== - сравнение точек на равенство

- - вычитание точек (возвращает вектор)

<< - вывод точки в поток

6. Статический метод
midpoint(const Point2D& p1, const Point2D& p2) - нахождение средней точки между двумя точками

7. Метод вывода состояния
printState() - вывод полной информации о состоянии точки
